# wordle-py_CSP
CSP Wordle game using turtle 1.0

## Colors:
### Dark theme:
| Where                                  | Color   |
| -------------------------------------- | ------- |
| Background:                            | black   |
| Foreground:                            | #121213 |
| Yellow (right letter, wrong position): | #b59f3b |
| Green (right letter and position):     | #538d4e |
| Not in word:                           | #3a3a3c |
| Blank                                  | #3a3a3c |
